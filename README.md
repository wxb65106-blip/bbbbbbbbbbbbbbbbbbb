## 🧩 第一部分：POW（Proof of Work）计算

### 文件：`pow_homework.py`

内容包含：
- 使用 SHA256 实现自己的 POW 挖矿函数
- 尝试找到满足 4 个 0 开头的 hash，并记录花费时间
- 尝试找到满足 5 个 0 开头的 hash，并记录花费时间
- 输出包括：
  - 昵称（bangbongo）
  - 找到的 nonce
  - 计算出的 hash
  - 耗时（秒）

示例输出（以我本机运行结果为例）：

耗时: 0.87 秒

## 🔐 第二部分：RSA 私钥签名 + 公钥验证

### 文件：`rsa_homework.py`

内容包含：

- 使用 Python 生成 RSA 公私钥对（2048 位）
- 用 **私钥** 对 POW 产生的 “昵称 + nonce” 进行签名  
- 输出签名（16 进制）
- 用 **公钥** 对签名进行验证  
- 如果签名正确 → 显示 **“签名有效”**
